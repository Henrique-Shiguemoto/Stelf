# STELF

Este é um projeto de prototipação de uma loja de jogos chamada de Stelf, que possui o intuito de facilitar a compra e venda de jogos propondo uma plataforma simples e confortável.

Existem dois tipos principais de usuários: 

- Cliente: Primordialmente realizam compras de jogos (outras operações são cadastrar contas, editar perfis, manipular carrinho de compras, acessar biblioteca, entre outras...)
- Desenvolvedoras: Disponibilizam jogos na loja (outras operações são cadastrar contas, editar suas contas, entre outras...)

# Tecnologias Utilizadas

- C# (Linguagem de Programação)
- WinForms (UI Framework)
- Visual Studio 2022 (IDE)
- MongoDB (Banco de Dados NoSQL)
